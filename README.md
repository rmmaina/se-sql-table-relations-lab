# SQL Table Relations Lab

## Overview
This project demonstrates advanced SQL querying using SQLite, focusing on JOINs, LEFT JOINs, subqueries, grouping, and aggregation using the Northwind-style CRM database.

## Tech Stack
- Python
- SQLite3
- Pandas
- Flask
- SQLAlchemy
- VS Code

## Features
- INNER JOIN and LEFT JOIN queries
- GROUP BY and HAVING usage
- Aggregate functions (COUNT, SUM, AVG)
- Subqueries for advanced filtering
- CAST usage for numeric sorting
- Multi-table relationships
